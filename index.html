Dharmik Mobile Application

Full Stack Application solution on both Andriod and IOS

The Dharmik project represents a comprehensive full-stack solution encompassing both Android and iOS mobile applications, underpinned by a robust backend hosted on Google Cloud Platform (GCP) with a self-managed MySQL database. Leveraging GCP for an optimized cost structure, Dharmik integrates a continuous CI/CD pipeline, ensuring agile delivery of features on a release-based model. This approach facilitates a dynamic connection with client feedback, driving enhancements with a commitment to innovation and long-term scalability.
